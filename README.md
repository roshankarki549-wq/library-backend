Library Management System
Project Overview

Library Management System is a web-based application developed to automate library operations such as book management, member management, book issuing, returning, and fine calculation.
The system helps librarians efficiently manage library resources and provides users with easy access to library services.

Features:
 -Authentication and Authorization
 -Admin Dashboard
 -Book Management (Add, Update, Delete Books)
 -Member Management
 -Issue and Return Books
 -Fine Calculation for Late Returns
 -Search Books
 -Notifications and Alerts
 -Responsive User Interface

Technology Stack:

Backend
 -Django
 -Django REST Framework
 -PostgreSQL
